# xmrigHelp

A smart, auto-updating script for [XMRig](https://xmrig.com/).  

## ✅ Features

- Auto-detects OS (Linux, macOS, Windows)
- Downloads extracts, and overwrites with the latest XMRig release

## 🚀 Usage

### Run locally:
```bash
python3 script.py <dir>
```

### Or run directly:

### Linux / macOS
```bash
curl -fL https://raw.githubusercontent.com/xn2UEPYak4R8/xmrigHelp/main/script.py | python3 - <dir>
```

Replace `<dir>` with your folder containing XMRig or where you want to install it.
